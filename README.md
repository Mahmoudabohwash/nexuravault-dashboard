# NexuraVault Dashboard
##  Live Demo
[View Live Site](https://nexuravault-dashboard.netlify.app/)
A dashboard for managing car sales and orders built with Vanilla JavaScript.

## Features
- Product and car inventory management
- Orders table with search and filtering
- Interactive charts using Chart.js
- Fully responsive design for mobile and desktop
- Lightweight and fast – no frameworks needed

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Chart.js
